Inner Content

Sphere theme has full support for the WordPress’s default Gutenberg editor.

With the help of Gutenber editor you can easily create various layouts and content you want, not only what you see on the demo site. In this section you will find what features are added to the Gutenberg editor and overview how content on the demo site is created.

If you are not familiar with Gutenberg editor, this video may help you:

3.1 Content

All content on the demo site is created with Gutenberg blocks.
Main blocks support settings for widths: Default width, Wide width, Full width.

Column layouts are created with the Columns block.

3.2 Extra styling

Sphere theme provides extra styles for some of the Gutenberg’s core blocks.

3.2.1 Headings

3.2.2 Subheadings

Subheadings are created with special button added to the Heading block.

There is also secondary style available, if the heading / subheading is bold.

3.2.3 Paragraphs

Extra styles for Paragraph block.

3.3 Reveal Effects

Reveal effects can be added to the content elements via particular block settings.

3.4 Parallax Images

For images with parallax effect on scroll, select the image’s style to Parallax.

3.5 Counter block

Sphere theme provides special Counter block, which can be added to the content.

3.6 Icons

Sphere theme is also bundled with icons which can be used with Heading blocks. Full list of icons with their respective CSS classes can be found here.

CSS class can be added in tab Advanced > Additional CSS field when editing the heading block.
