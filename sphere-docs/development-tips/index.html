Development Tips

8.1 Child Theme

If you wish to modify the themeâ€™s code, it is the highly recommended to do it via child theme. Your changes wonâ€™t be lost when updating the theme. Child theme is included in the download package.
More info about Child Themes here.

In case of just cosmetic changes you can use the Additional CSS field in Appearance > Customize.

8.2 Theme Fonts

All theme fonts and sizes are set in functions.php file as CSS variables.
So they can be easily adjusted via child theme.

--heading-font: "Montserrat", sans-serif;
--heading-font-weight: 800;

--xxl-font-size: 7.4375rem;
--xxl-line-height: 8.5rem;

--xl-font-size: 5.5625rem;
--xl-line-height: 6.5rem;

--h1-font-size: 4.1875rem;
--h1-line-height: 5rem;

--h2-font-size: 3.125rem;
--h2-line-height: 4rem;

--h3-font-size: 2.3125rem;
--h3-line-height: 3rem;

--h4-font-size: 1.75rem;
--h4-line-height: 2.5rem;

--h5-font-size: 1.3125rem;
--h5-line-height: 2rem;

--h6-font-size: 1rem;
--h6-line-height: 1.75rem;

--xs-font-size: 0.75rem;
--xs-line-height: 1.5rem;

--subheading-font: "Montserrat", sans-serif;
--subheading-font-weight: 700;
--subheading-transform: uppercase;
--subheading-spacing: 0.25em;

--xxl-subheading: 1.125rem;
--xl-subheading: 1rem;
--h123-subheading: .9375rem;
--h456-subheading: .8125rem;
--xs-subheading: .75rem;

--buttons-font: "Montserrat", sans-serif;
--buttons-font-weight: 700;

--body-font: "Raleway", sans-serif;
--body-font-size: 1.0625rem;
--body-line-height: 2.1;

When changing fonts, you may want to fine-tune the heading’s underline in custom CSS:

#content .is-style-underlined span,
#content .is-style-underlined-xl span {
    background-position: 0 .86em;
    background-size: 100% .175em;
}

8.3 AJAX

For links without ajax, you can use .no-ajax css class.

As the theme is ajax powered, it can cause some problems with 3rd party plugins which use ajax / javascript as well. If your plugin doesnâ€™t work properly, you can re-init the plugin after ajax is completed with help of the snippet below.

jQuery(document).ajaxComplete(function() {
    // Re-init your plugin here...
});
